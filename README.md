# FRONT END IP MANAGER APP
- a typescript based vite app for ip manager.
- react version 19

# build frontend
- make sure overall installation done first
- cd to ipmanager
- open git bash
- ./frontend.sh
- will create .env from .env.example
- will build FE
- accessible through http://localhost

# dev
- npm run start
- make sure .env files are created from .env.example
- refer to build instructions
